# Data Notebooks for SignalK

## ALPHA - Use with care

Data notebooks, using [Marimo](https://marimo.io) and Python for DAG aware notebooks. Notebooks
run entirely in the browser, using WebAssembly (WASM) to keep the server load minimal and best
suited to Raspberry Pi, NanoPi etc servers.

Intention is to have these wired up by default to SignalK History API.
